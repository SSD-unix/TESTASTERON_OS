Forged in Silicon

A bare-metal x86 operating system built from scratch in C and Assembly. No bloat. No abstractions. Absolute hardware control.

Launch Web Terminal

Core Architecture

Custom Kernel

Monolithic architecture managing the Global Descriptor Table (GDT) and Interrupt Descriptor Table (IDT). Direct access to CPU registers.

Low-Level I/O

Custom disk drivers reading directly from sectors. Interfacing with hardware via explicit IN/OUT port instructions.

Integrated Shell

A built-in command-line interface interpreting user input, managing keyboard IRQs, and pushing characters straight to the VGA text buffer.

The Ecosystem

Building an OS is just the beginning. The vision extends to advanced ventures and specialized tooling.

TURBOGET

Python 3 API Aviation

An auxiliary project developed alongside the OS. TURBOGET is a highly specialized tracking application designed to monitor L-410 Turbolet aircraft in real-time by hooking into aviation data pipelines.
